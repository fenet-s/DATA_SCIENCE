# Python Foundations Assignment

This repository contains my solutions for the Python Foundations Daily Assignment. As an aspiring Data Scientist, I have focused on writing clean, modular, and optimized code.

## Implementations
1. **Level 1: Prime Checker** - Uses an optimized O(sqrt(n)) algorithm.
2. **Level 2: Fibonacci Sequence** - Generates a dynamic list of n-terms.
3. **Level 3: CLI Calculator** - A modular approach using functional mapping.

## How to Run
Ensure you have Python 3.x installed. Run each script using:
`python filename.py`
